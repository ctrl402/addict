中世纪（5–15世纪）
酿造由家庭转向修道院作坊；修士改进工艺并引入啤酒花（约公元8世纪），啤酒花逐步取代格鲁特（Gruit）成为苦味与防腐剂，奠定现代啤酒基础。
1516年
巴伐利亚颁布《啤酒纯酿法令》（Reinheitsgebot），规定仅用水、大麦芽、啤酒花（后加酵母）酿造——世界最古老食品法规之一。
19世纪
工业革命推动规模化：蒸汽糖化锅、制冷机（Linde，1873）、巴氏杀菌（Pasteur，1860s）、纯种酵母培养（Hansen，1883）。圆柱圆锥发酵罐（CCV）与底面发酵拉格普及。
20–21世纪
全球连片化大生产、PLC/DCS自动化控制；2010年代起精酿运动兴起，低醇/无醇、果味、个性化成为新赛道。

2综述：工业化啤酒全流程制作过程

工业化啤酒生产以大麦芽、大米/玉米（辅料）、啤酒花、水、酵母为原料，经制麦—糖化—过滤—煮沸—冷却—发酵—储酒—过滤—灌装—杀菌—检验等单元构成连续生产链。各单元均对应"三传一反"中的特定传递与反应过程。

图2-1 工业化啤酒生产全流程（黄色=相变/反应关键单元，绿色=传热单元，蓝色=物理分离单元）

2.1 原料筛选

原料作用质量要求对应传递/反应
大麦芽淀粉+酶源，色香主体糖化力>250°WK，浸出率>80%反应（酶水解）
大米/玉米辅料，提供可发酵糖，清爽口感淀粉>75%，脂肪低反应（糊化）
啤酒花苦味（α-酸异构）、香气、防腐α-酸5–15%，贮藏指数<0.4反应（异构化）
水占成品90%+，影响风味硬度适中，余氯=0质量传递介质
酵母乙醇发酵核心微生物纯种、活力>95%反应（生物催化）

2.2 粉碎与糖化

麦芽粉碎（对辊/湿法）控制皮壳完整以提高过滤效率。糖化在62–68℃下利用麦芽自身α-淀粉酶与β-淀粉酶将淀粉水解为麦芽糖、麦芽三糖、葡萄糖等可发酵糖；辅料需预先糊化（70–90℃）。

2.3 过滤麦汁

糖化醪经过滤槽（Lauter tun）利用麦皮作天然滤层，得澄清麦汁；洗糟水（76–78℃）回收残糖。质量传递+过滤分离。

2.4 加酒花煮沸与回旋除杂

麦汁煮沸（100℃，60–90 min）加入酒花：α-酸异构化为异α-酸（苦味），挥发不良风味，凝固蛋白质（热凝固物）。回旋沉淀槽（Whirlpool）离心分离热凝固物——动量传递（旋流）。

2.5 冷却与接种

薄板换热器将热麦汁从~95℃冷至发酵温度（拉格8–10℃，艾尔15–20℃），同时充氧至8–12 mg/L供酵母增殖——热量传递+气液传质。

2.6 主发酵

酵母在圆柱圆锥发酵罐（CCV）内将糖转化为乙醇与CO₂，同时生成酯类、高级醇等风味物质。主发酵5–10天，分迟缓期、对数期、稳定期。核心反应工程单元（详见第3–6章）。

2.7 低温储酒（熟化）

降温至0–2℃储酒7–14天（拉格源自德语 Lager=储藏），酵母沉降至锥底，双乙酰还原、风味成熟、CO₂饱和。热量传递+相分离。

2.8 过滤与灌装

硅藻土/纸板精滤至浊度<0.5 EBC。灌装采用等压灌装（先充CO₂背压，避免溶氧与泡沫），玻璃瓶/易拉罐/Keg桶。动量传递+多相传质。

2.9 巴氏杀菌与检验入库

隧道式巴氏杀菌（60–62℃，15–20 min，PU值15–30）灭活残留酵母与杂菌。检验理化指标（酒精度、原麦汁浓度、色度、苦味值BU、CO₂）、微生物合格后入库。

三传一反映射：糖化=反应（酶催化）；过滤/回旋=动量传递；煮沸/冷却/储酒=热量传递；充氧/CO₂饱和=质量传递；发酵=生物化学反应。整个流程正是"三传一反"的集成工程。

3.3 反应动力学机理

① 比生长速率 — Monod方程 + Luong产物抑制

$$\mu = \mu_{\max}\cdot\frac{S}{K_s + S}\cdot\left(1-\frac{P}{P_{\max}}\right)^n,\quad PMonod J. (1949) Ann Rev Microbiol 3:371-394；产物抑制项见 Luong JHT (1985) Biotechnol Bioeng 27:280-285
符号定义单位典型值
μ酵母比生长速率h⁻¹0.02–0.06
μ_max最大比生长速率（温度相关）h⁻¹0.04（10℃）
S限制性底物（可发酵糖）浓度g/L0–120
K_sMonod半饱和常数g/L0.1–1.0
P产物（乙醇）浓度g/L0–80
P_max酵母最大耐受乙醇浓度g/L80–110
n产物抑制指数—1

② 细胞生长与衰亡

$$\frac{dX}{dt} = \mu\cdot X - k_d\cdot X$$
X为酵母细胞浓度(g/L)；k_d为衰亡系数(h⁻¹)，反映营养耗尽与乙醇毒性下的细胞死亡

③ 底物消耗 — Luedeking-Piret形式

$$\frac{dS}{dt} = -\frac{1}{Y_{X/S}}\mu X - m_s X$$
Luedeking R, Piret EL (1959) J Biochem Microbial Technol Eng 1:393-412。Y_{X/S}为菌体对底物表观得率(g/g)，m_s为维持系数 g/(g·h)

④ 产物生成 — Luedeking-Piret模型

$$\frac{dP}{dt} = (\alpha\mu + \beta)\cdot X$$
α为生长相关产物得率系数(g/g)，β为非生长相关产物得率系数(h⁻¹)。啤酒发酵属于混合型：生长偶联+非生长偶联并存

⑤ CO₂生成（化学计量）

$$\frac{dCO_2}{dt} = Y_{P/CO_2}\cdot\frac{dP}{dt},\quad C_6H_{12}O_6\rightarrow 2C_2H_5OH + 2CO_2$$
理论 Y_{P/CO₂} = M(CO₂)/M(EtOH) = 88/92 ≈ 0.957

⑥ 温度效应 — Arrhenius修正

$$\mu_{\max}(T) = \mu_{\max,ref}\cdot\exp\left[-\frac{E_a}{R}\left(\frac{1}{T}-\frac{1}{T_{ref}}\right)\right]$$
E_a表观活化能(J/mol)；R=8.314 J/(mol·K)；T_ref参考温度(K)。生物反应典型E_a=40–80 kJ/mol

3.4 传质机理 — 气液氧传质

$$\frac{dC_L}{dt} = k_La\,(C_L^* - C_L) - q_{O_2}\cdot X$$
k_La为体积氧传质系数(h⁻¹)，C_L*为饱和溶氧，q_{O₂}为比耗氧速率。接种前充氧至8–12 mg/L以支撑酵母对数生长
教学要点：Monod方程描述底物限制下微生物生长（对应"一反"中反应动力学）；Luong项体现产物抑制（乙醇积累反馈抑制）；Luedeking-Piret将底物消耗与产物生成解耦为生长相关与非生长相关——这是生化反应工程区别于一般化学反应工程的精髓。

4建模：工业级发酵过程动态建模与灵敏度分析

4.1 ODE系统模型

将第3章机理方程组装为四元常微分方程组，描述酵母X、底物S、产物P、CO₂随时间演化：

$$\begin{cases}\dot X = \mu(S,P,T)\,X - k_d X\\[3pt]\dot S = -\dfrac{1}{Y_{X/S}}\mu X - m_s X\\[6pt]\dot P = (\alpha\mu+\beta)X\\[3pt]\dot{\dot{CO_2}} = Y_{P/CO_2}\,\dot P\end{cases}$$

4.3 数值求解（RK45）

采用 scipy.integrate.solve_ivp（显式Runge-Kutta，rtol=1e-8）在0–168 h区间积分。下方为模型核心代码片段：

def model_odes(t, y, p):
    X, S, P, CO2 = y
    # Monod + Luong产物抑制
    mu = p['mu_max'] * S/(p['Ks']+S) * max(0, 1-P/p['Pmax'])**p['n']
    mu *= arrhenius_factor(T, p)  # 温度修正
    dX  = mu*X - p['kd']*X
    dS  = -mu*X/p['Yxs'] - p['ms']*X  # 底物耗尽时置零
    dP  = (p['alpha']*mu + p['beta'])*X
    dCO2= p['Ypco2']*dP
    return [dX, dS, dP, dCO2]

sol = solve_ivp(model_odes, [0,168], y0, args=(p,), method='RK45', rtol=1e-8)

关键结论：

1. P_max（产物耐受）最敏感——选育耐高乙醇酵母菌株是提升发酵效率的根本途径。

2. Y_{X/S}（菌体得率）影响显著且非对称——得率降低会"浪费"碳源于菌体而非产物，需控制接种量与溶氧。

3. μ_max与温度T强耦合——温度升高加速发酵但风味劣化，体现经济性与品质的权衡（见第6章）。

4. K_s几乎无影响——说明工业浓度远超半饱和常数，处于零级动力学区，底物不构成限制。

5控制：工业级发酵批次过程控制系统分析

5.1 批次过程特点

啤酒发酵是典型的批次（间歇）过程：物料一次性投入，状态随时间演化，无稳态操作点。控制目标是在保证风味品质前提下，按预设温度-时间轨线完成发酵，缩短周期。

特征表现控制挑战
时变非线性μ、q_p随S、P、T变化需轨迹跟踪而非定值控制
大时滞酵母代谢响应慢（小时级）反馈控制需预测补偿
关键变量不可测糖、乙醇在线难测需软测量/状态估计
重复性好批次间工艺固定适合批次间迭代学习控制

5.2 控制系统架构

图5-1 啤酒发酵多层级控制系统架构：计划→批次管理→APC优化→DCS基础回路→被控对象

5.3 温度轨线跟踪控制

主发酵温度需按工艺轨线变化（如迟缓期10℃→对数期12℃→还原期14℃→降温0℃）。采用串级PID+前馈：

$$u(t) = K_p\left[e(t) + \frac{1}{T_i}\int_0^t e\,d\tau + T_d\frac{de}{dt}\right] + K_f\,d(t)$$
e(t) = T_sp(t) - T(t) 为温度偏差；d(t)为发酵放热前馈量；K_f为前馈增益

5.4 软测量与状态估计

糖度与乙醇浓度难以在线测量，用扩展卡尔曼滤波（EKF）融合离线化验与模型预测：

$$\hat{x}_{k|k} = \hat{x}_{k|k-1} + K_k\,(z_k - h(\hat{x}_{k|k-1}))$$
x=[X,S,P]ᵀ状态向量；z_k为离线化验（糖度/酒精度，每8h一次）；K_k为卡尔曼增益

5.5 批次间迭代学习控制（ILC）

利用批次重复性，将上一批误差用于修正本批设定值，逐步收敛：

$$u_{j+1}(t) = u_j(t) + \Gamma\, e_j(t)$$
j为批次索引；Γ为学习增益矩阵。批次间渐进消除温度轨线跟踪误差
控制工程要点：发酵控制体现了层级化、模型驱动、数据融合的现代过程控制思想——基础PID保稳定，APC/MPC求最优，ILC利用批次重复性持续改进，软测量补足在线感知缺口。

6优化：品质-效率-经济性的系统优化

6.1 优化问题表述

$$\min_{T(t),\,X_0,\,t_f}\; J = \underbrace{w_1\frac{t_f}{t_{f,ref}}}_{\text{效率}} + \underbrace{w_2\frac{C_{energy}}{C_{ref}}}_{\text{经济}} + \underbrace{w_3\,F_{defect}}_{\text{品质惩罚}}$$
约束：T∈[8,16]℃；双乙酰≤0.1 mg/L；酯类/高级醇在风味窗内；残糖≤2 g/L；P∈[35,45]g/L

6.2 优化变量与策略

决策变量优化方向对品质影响对效率/经济影响
主发酵温度T(t)↑加速但控上限过高→高级醇↑、酯失衡↑5℃周期缩短~30%
接种量X₀↑缩短迟缓期过高→酵母自溶味↑0.5 g/L周期-8%
溶氧DO适度供给过高→氧化味、酯↓充氧能耗
压力(加压发酵)0.1–0.2 MPa抑制酯生成，CO₂饱和快减少CO₂回收成本
降温速率快降省时过快→酵母应激双乙酰↑冷媒能耗↑

6.4 工业优化实践

🌡️

变温轨线优化

用动态规划求解最优T(t)轨线，在对数期适度升温缩短周期，还原期升温促双乙酰还原，末期快降温。较恒温节能12–18%。

🧬

菌株工程

选育高P_max菌株（第4章灵敏度首位），耐高乙醇且风味稳定，直接提升发酵上限与效率。

♻️

热集成与CO₂回收

发酵放热→预热清洗水；CO₂回收净化回用灌装背压，降低采购成本，全厂水耗<3 L/L啤酒。

6.5 数字化与AI优化

现代啤酒厂部署数字孪生：实时发酵模型+在线传感+EKF状态估计，结合历史批次数据训练LSTM/Transformer预测发酵终点与风味指标，实现"预测-决策-执行"闭环，单批次综合成本下降5–10%。

系统优化要点：以品质为硬约束（风味窗、残糖、酒精度），在效率（发酵周期）与经济性（能耗、原料、CO₂）间求Pareto平衡。第4章灵敏度分析已指明优化杠杆优先级：菌株（P_max）> 得率（Y_{X/S}）> 温度-速率协同。
